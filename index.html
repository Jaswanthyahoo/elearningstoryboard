FAQ's

What is CAE?

Basically, CAE is a method used by engineers to understand how elements in a product would react to different forces being exerted on them. Today, designs are far more complex for an engineer to apply traditional mathematical equations. For instance, how would a building react to different forces being exerted by wind on it, how would parts in an engine behave- will different shapes or thickness have any difference? To solve these, it requires a methodology, i.e., geometric model to replicate the prototype and analyzed with Finite analysis method with the computer Tools.
